# DeePMD-kit

## 安装

### 通过Docker安装

一种方便快捷的安装方法即通过现有的docker镜像进行安装。当然这需要实现配置好docker环境和nvidia-docker。

```bash
$ docker pull cloudac7/deepmd-kit:cuda-9.0-ssh-ubuntu16.04
```
